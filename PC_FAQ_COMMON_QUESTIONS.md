# PC 조립 자주 묻는 질문 (FAQ) 💡

## 📋 실제 사용자들이 자주 묻는 질문 모음

### 🖥️ CPU 관련

#### Q1. CPU를 선택할 때 코어 수와 클럭 속도 중 뭐가 더 중요한가요?
**A**: 용도에 따라 다릅니다!
- **게임**: 클럭 속도가 더 중요 (6코어 이상 권장)
- **영상편집/렌더링**: 코어 수가 더 중요 (8코어 이상 권장)
- **사무/웹서핑**: 4코어면 충분

#### Q2. 인텔과 AMD 중 어떤 걸 선택해야 하나요?
**A**: 
- **인텔**: 게임 최적화가 잘 됨, 안정성 우수
- **AMD**: 가성비 우수, 멀티태스킹 강함
- **추천**: 예산과 용도에 따라 선택

#### Q3. CPU 쿨러는 정품 쿨러로 충분한가요?
**A**:
- **정품 쿨러로 충분**: 저전력 CPU (65W 이하), 오버클럭 안 함
- **별도 쿨러 필요**: 고성능 CPU (95W 이상), 오버클럭, 소음 민감
- **추천**: 3만원대 타워형 쿨러면 충분

#### Q4. F 시리즈 CPU(예: 14600KF)와 일반 버전 차이는?
**A**:
- **F 시리즈**: 내장 그래픽 없음, 가격 저렴
- **일반 버전**: 내장 그래픽 있음, 약간 비쌈
- **결론**: 별도 그래픽카드 구매할 거면 F 시리즈가 가성비 좋음

---

### 🎮 그래픽카드 관련

#### Q5. 그래픽카드 VRAM은 몇 GB가 적당한가요?
**A**:
- **1080p 게임**: 6~8GB (RTX 4060, 4060 Ti)
- **1440p 게임**: 8~12GB (RTX 4070)
- **4K 게임/영상작업**: 12GB 이상 (RTX 4070 Ti, 4080)

#### Q6. 중고 그래픽카드 사도 되나요?
**A**:
- **채굴용 중고**: ❌ 비추천 (수명 짧음)
- **게임용 중고**: ⚠️ 신중하게 (보증 기간, 판매자 신뢰도 확인)
- **새 제품**: ✅ 추천 (AS, 안정성)

#### Q7. RTX와 GTX 차이는 뭔가요?
**A**:
- **RTX**: 레이트레이싱 지원, DLSS 3.0, 최신 기술
- **GTX**: 구형, 가성비는 좋으나 신기술 미지원
- **추천**: 2024년 이후는 RTX 권장

### 💾 RAM 관련

#### Q8. RAM은 몇 GB가 필요한가요?
**A**:
- **8GB**: ❌ 2024년 기준 부족
- **16GB**: ✅ 게임, 일반 작업 충분
- **32GB**: ✅✅ 게임+방송, 영상편집 권장
- **64GB**: 전문 작업 (3D, 렌더링)

#### Q9. DDR4와 DDR5 중 어떤 걸 선택해야 하나요?
**A**:
- **DDR4**: 가성비 우수, 호환성 좋음
- **DDR5**: 성능 우수, 최신 플랫폼 (Intel 12세대+, AMD 7000번대+)
- **추천**: 신규 조립은 DDR5 권장

#### Q10. RAM 클럭(MHz)은 얼마나 중요한가요?
**A**:
- **DDR4**: 3200MHz 이상 권장
- **DDR5**: 5600MHz 이상 권장
- **효과**: 체감 성능 차이는 크지 않음 (5~10%)

#### Q11. RAM은 2개(듀얼채널)가 1개보다 좋나요?
**A**: ✅✅ 매우 중요!
- **듀얼채널 (8GB x2 = 16GB)**: 성능 최대 50% 향상
- **싱글채널 (16GB x1)**: 비추천
- **결론**: 항상 2개 이상 구매 권장

### 💿 저장장치 관련

#### Q12. SSD와 HDD 중 어떤 걸 선택해야 하나요?
**A**:
- **SSD**: 빠름, 조용함, 내구성 우수 → **OS 설치 필수**
- **HDD**: 저렴, 대용량 → 데이터 저장용
- **권장 구성**: SSD 500GB (OS/프로그램) + HDD 2TB (데이터)

#### Q13. NVMe SSD와 SATA SSD 차이는?
**A**:
- **NVMe**: 읽기 5000MB/s 이상, M.2 슬롯, 빠름
- **SATA**: 읽기 550MB/s, 2.5인치, 느림
- **체감 차이**: 게임/일반 사용은 크지 않음
- **추천**: 가격 비슷하면 NVMe 선택

#### Q14. SSD 용량은 얼마나 필요한가요?
**A**:
- **256GB**: ❌ 2024년 기준 부족
- **500GB**: ✅ OS + 주요 프로그램
- **1TB**: ✅✅ OS + 게임 여러 개 (권장)
- **2TB**: 게임 많이 하는 경우

### ⚡ 파워 관련

#### Q15. 파워 용량은 어떻게 계산하나요?
**A**: 
**간단 계산법**:
```
총 소비 전력 = CPU TDP + GPU TDP + 100W (기타)
파워 용량 = 총 소비 전력 × 1.5배
```

**예시**:
- RTX 4060 (115W) + i5-14600K (125W) + 100W = 340W
- 권장 파워: 500~650W

#### Q16. 80 PLUS 인증이 뭔가요?
**A**:
- **Bronze**: 효율 80% 이상 (저렴)
- **Gold**: 효율 87% 이상 (권장)
- **Platinum**: 효율 90% 이상 (고급)
- **Titanium**: 효율 92% 이상 (프리미엄)
- **추천**: Gold 이상 권장

#### Q17. 파워는 정격출력을 봐야 하나요?
**A**: ✅ 반드시!
- **정격출력**: 지속적으로 공급 가능한 전력 (중요)
- **최대출력**: 순간 최대치 (의미 없음)
- **주의**: 중국산 저가 파워는 정격 허위 표시 많음

### 🏠 케이스 관련

#### Q18. 케이스 크기는 어떻게 선택하나요?
**A**:
- **미니타워**: 작음, 확장성 제한, ITX/M-ATX 지원
- **미들타워**: ✅✅ 가장 일반적, ATX 지원 (권장)
- **풀타워**: 큼, 확장성 최고, E-ATX 지원

#### Q19. 케이스 선택 시 뭘 봐야 하나요?
**A**: 체크리스트
- ✅ 그래픽카드 최대 길이 (330mm 이상 권장)
- ✅ CPU 쿨러 최대 높이 (160mm 이상 권장)
- ✅ 파워 위치 (하단 권장)
- ✅ 먼지 필터 유무
- ✅ 통풍 (메쉬 패널 > 강화유리)

### 🔧 호환성 관련

#### Q20. CPU와 메인보드 소켓이 맞아야 하나요?
**A**: ✅ 매우 중요!
- **Intel**: LGA1700 (12~14세대), LGA1200 (10~11세대)
- **AMD**: AM5 (7000번대), AM4 (5000번대 이하)
- **주의**: 소켓 안 맞으면 장착 불가

#### Q21. 메인보드 크기는 어떻게 선택하나요?
**A**:
- **Mini-ITX**: 17x17cm, 확장성 최소
- **M-ATX**: 24x24cm, 적당한 확장성
- **ATX**: 30x24cm, ✅ 가장 일반적 (권장)
- **E-ATX**: 30x27cm, 확장성 최대

#### Q22. RAM은 몇 MHz까지 지원하나요?
**A**: 메인보드 스펙 확인 필수!
- **Intel 13~14세대**: DDR5 7200MHz 이상 지원
- **AMD 7000번대**: DDR5 6000MHz 권장 (EXPO)
- **주의**: CPU와 메인보드 모두 지원해야 함

### 🌡️ 쿨링 관련

#### Q23. 수냉 쿨러와 공랭 쿨러 중 뭐가 좋나요?
**A**:
- **공랭 쿨러**: 저렴, 고장 적음, 소음 적음 → ✅ 일반 사용자 권장
- **수냉 쿨러**: 비쌈, 쿨링 우수, 디자인 좋음 → 오버클럭, 고성능

#### Q24. 케이스 팬은 몇 개가 적당한가요?
**A**:
- **최소**: 전면 흡기 1개, 후면 배기 1개
- **권장**: 전면 흡기 2개, 후면 배기 1개, 상단 배기 1개
- **팁**: 흡기 > 배기 = 양압 (먼지 적음)

### 🔨 조립 관련

#### Q27. PC 조립 순서가 어떻게 되나요?
**A**:
1. 메인보드에 CPU 장착
2. CPU 쿨러 장착
3. RAM 장착 (2번, 4번 슬롯 우선)
4. 메인보드를 케이스에 장착
5. 파워 장착 및 케이블 연결
6. 그래픽카드 장착
7. SSD/HDD 장착
8. 케이블 정리

#### Q28. PC 조립 후 바이오스 설정은 어떻게 하나요?
**A**: 필수 설정
1. **XMP/EXPO 활성화** (RAM 성능 최대화)
2. **부팅 순서 설정** (SSD 우선)
3. **팬 속도 조절** (소음 최적화)

### 🆕 업그레이드 관련

#### Q29. 기존 PC 업그레이드 시 뭘 먼저 바꿔야 하나요?
**A**: 우선순위
1. **SSD 추가/교체** (체감 성능 향상 최대)
2. **RAM 증설** (8GB → 16GB)
3. **그래픽카드 업그레이드** (게임 성능 향상)
4. **CPU/메인보드** (마지막, 가장 비쌈)

#### Q30. 몇 년마다 PC를 바꿔야 하나요?
**A**:
- **게임용**: 3~5년 (그래픽카드는 2~3년마다)
- **사무용**: 5~7년
- **팁**: 부품별 업그레이드로 수명 연장 가능

```sql
CREATE TABLE pc_faq (
    id BIGINT AUTO_INCREMENT PRIMARY KEY,
    question VARCHAR(500) NOT NULL,
    answer TEXT NOT NULL,
    category VARCHAR(50),
    keywords VARCHAR(255),
    view_count INT DEFAULT 0,
    helpful_count INT DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    INDEX idx_category (category),
    INDEX idx_keywords (keywords),
    FULLTEXT INDEX idx_question (question)
```
